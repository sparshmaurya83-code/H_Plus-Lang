# H+ (Hyper+) – Simpler than speaking  

**H+** is a tiny, fast, beginner-friendly programming language experiment built in Rust.  
Goal: make coding feel more like natural language with keywords like `say`, `when`, `use`, etc.

> Note: This is an early MVP / learning project — not production-ready yet, but fun to play with!

## Quick Demo

```powershell
# Clone & build (needs Rust installed)
git clone https://github.com/sparshmaurya83/H_plus-lang.git
cd H_plus-lang
cargo build --release

# Run the example
cargo run -- run examples/hello.h+
